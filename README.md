# file-integrity-checker
Verificador de Integridade de Arquivos é uma ferramenta de linha de comando que calcula hashes SHA-256 de arquivos de log para detectar alterações não autorizadas. Na primeira execução, ela registra o estado original dos arquivos e, nas próximas verificações, compara os hashes atuais com os armazenados para identificar possíveis adulterações.
